Средство проверки орфографии "Спеллчекер"

Описание
Данное приложение - реализация проверки орфографии на русском языке.

Требования
fuzzywuzzy версии 0.17.0
python_Levenshtein версии 0.12.0(необязательно)

Состав
Консольная версия: main_spellchecker.py
Вспомогательные модули для консольной версии:
file_spellchecker.py(работа с файлами)
generate_base.py(генерация базы по файлу)
interactive_spellchecker.py(интерактивный режим)
stemmer.py("стемминг" слова-приведение словоформ к общей основе)
Разделение слитного текста пробелами:put_spaces.py
Тесты:tests_leva\
База данных со всеми словарями:speller.db
Консольная версия(единственная)
Справка по запуску:.\main_spellchecker.py -h[--help]

usage: main_spellchecker.py [--put_spaces] [-h] [--imode] [--ubase UBASE] [--dict DICT] [--file FILE [FILE ...]] [-j]

optional arguments:
  --put_spaces, -ps     Интерактивный режим разделения текста по пробелам
  -h, --help            Показать это сообщение и закончить работу
  --imode               Запуск в интерактивном режиме
  --ubase UBASE         Использование данного файла для генерации базы(словаря)
  --dict DICT           Выбор уже сгенерированного словаря, отличного от стандартногоВозможные словари: virtualdict
  --file FILE [FILE ...], -f FILE [FILE ...]
                        Запуск в пакетном режиме(для работы с файлами) Введите путь к файлу, который нужно
                        проанализировать и путь к файлу для помещения результата По умолчанию создается новый файл
                        res.txt
  -j                    В сочетании с --file(передается как отдельный аргументНЕ после --file или может образовывать
                        префикс -jf) позволяет пропустить шаги взаимодействия с программой. Текст будет без изменений
                        записан в результирующий файл, но в конце будут добавлены данные об ошибках и возможные
                        варианты

На данный проект написаны тесты, их можно найти в tests_leva\.
